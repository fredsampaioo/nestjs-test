- para rodar localmente a aplicação de back (porta 4000):
```
cd backend
yarn start
```

- para rodar localmente a aplicação de front (porta 4001):
```
cd frontend
yarn start
```

- para rodar o backend via docker, caso nao tenha mongo instalado:
```
cd backend
yarn start
```
